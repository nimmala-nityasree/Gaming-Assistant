# 🎮 GamerGPT – AI Gaming Assistant

GamerGPT is an AI-powered gaming assistant built using Python and Google's Gemini API. It helps gamers discover new games, compare titles, check PC requirements, get gaming tips, and receive personalized recommendations.

## 🚀 Features

* 🎯 Game Recommendations
* 🖥️ PC Requirement Analysis
* ⚔️ Game Comparisons
* 💡 Gaming Tips & Tricks
* 🎮 Personalized Game Suggestions
* 💬 Interactive Chat Interface

## 🛠️ Technologies Used

* Python
* Google Gemini API
* Google GenAI SDK

## ⚙️ Installation

### Clone the Repository

```bash
git clone <your-repository-link>
cd GamerGPT
```

### Install Dependencies

```bash
pip install google-genai
```

## 🔑 Configure API Key

Open `gamer_assistant.py` and add your Gemini API key:

```python
API_KEY = "YOUR_API_KEY"
```

## ▶️ Run the Project

```bash
python gamer_assistant.py
```
